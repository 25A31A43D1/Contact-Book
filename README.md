Contact Book Management System

A simple Contact Book Management System built using C++.
This project allows users to store and manage contact details using file handling.

Features
Add new contacts
View all contacts
Search contacts by name
Delete contacts
Save contacts permanently using file handling
Technologies Used
C++
File Handling
Object-Oriented Programming
Project Structure
ContactBook/
│
├── main.cpp
├── contacts.txt
└── README.md
How to Run
Compile
g++ main.cpp -o contactbook
Run
./contactbook

For Windows:

contactbook.exe
Menu Options
1. Add Contact
2. View Contacts
3. Search Contact
4. Delete Contact
5. Exit
